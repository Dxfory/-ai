# Fenran Technique Templates

## Implemented V1 Template

Template id: fenran-basic-three-layer-v1

Layers:

1. base_color: masked, light apparent-color wash.
2. layer_01: first light wash.
3. layer_02: second deepening.
4. layer_03: local reinforcement.
5. final_teaching_preview: line draft over cumulative color.

## Technique Graph

The renderer writes technique_graph.json for each sample. It records layer ids, modes, opacities, template version, and review flag.

## Current Rule Limits

- Does not yet distinguish front leaf, back leaf, petal, bud, branch, or insect as separate templates.
- Does not yet infer root-to-tip gradient direction.
- Does not claim real pigment-layer truth.
