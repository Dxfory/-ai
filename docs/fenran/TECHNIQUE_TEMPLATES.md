# Fenran Technique Templates

Current plan version: `fenran-teaching-plan-v2`.

1. Optional base color, executed only when the user selects it.
2. First fenran: 花青加淡墨, light and transparent, establishing structural light and shade.
3. Deepen fenran: only 花青, continuing from the first stage and strengthening turns, overlaps, veins, recesses, and dark structure.
4. Sap-green glaze: 正叶整体罩染汁绿, preserving the established modeling and highlights.

Every stage is a complete source-proportion image. The image model is not responsible for headings, numbering, color cards, borders, multiple panels, or lesson-page composition.
