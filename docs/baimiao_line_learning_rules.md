# 白描线稿学习规则

## 核心原则

白描稿不是边缘检测图，也不是 AI 重新设计的线稿。它必须学习原作中已经存在的作者勾线和教材白描稿里的线条逻辑。

生成白描稿时，优先级如下：

1. 原画里可见的墨线、轮廓线、叶脉线、花瓣边线、枝干线、鸟体结构线。
2. 教材中对应白描稿已经明确画出的线。
3. 教材文字中说明的勾线、质感线、转折线、遮挡线。
4. 形理逻辑推断出的必要连接关系。

第 4 条只能用于解释和轻微补足，不能改变对象数量、构图方向、枝叶走势或主体位置。

## 需要学习的线

- 骨架线：枝干走向、主叶脉、花果鸟体的主要结构。
- 轮廓线：花瓣、叶片、果实、鸟体外形。
- 转折线：叶片翻转、果皮开裂、花瓣卷折、羽毛分组。
- 包裹线：果皮包住籽粒、花瓣包住花心、叶片压住枝条。
- 遮挡线：前后关系明确的断线、续线、压叠线。
- 质感线：石榴籽、瓜纹、羽毛、叶脉、虫翅脉等。
- 虚实线：笔断意连、局部减弱、疏密节奏。

## 不应该学习的线

- 照片光影边缘。
- 纸张纹理、扫描阴影、书脊阴影。
- AI 为了好看新加的叶脉、枝条、花瓣、鸟羽。
- 与原作构图不一致的重新组织线。
- 把设色色块边缘误认为必须勾出的墨线。

## 白描稿验收

- 构图方向不变。
- 对象数量不变。
- 主枝走势不变。
- 叶片/花瓣/果实/鸟体位置不变。
- 原作已有重要墨线尽量保留。
- 纹理线可以简化，但不能把结构关系抹掉。
- 输出黑白清楚，少灰度、少背景噪声。
